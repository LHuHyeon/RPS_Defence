+ Unity 2D Defence 게임입니다.

+ RPS는 가위바위보(Rock-Paper-Scissors)의 약자로 가위바위보 카드를 뽑아 용병을 획득하고 제한시간 안에 몬스터를 처치하는 모바일 Defnece 게임입니다.

+ 현재 Repository에는 유료에셋 사용으로 인해 소스코드만 등록되어 있습니다.

+ 개발기간: 2023.09.09 ~ 2023.11.20 ( 약 2개월 )

+ 형상관리: Git SourceTree
